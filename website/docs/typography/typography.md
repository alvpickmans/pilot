title: Typography System
---

Overview
- Primary industrial font: DIN 1451 (or licensed clone)
- Secondary monospace: IBM Plex Mono or JetBrains Mono

Sizing & rhythm
- Base grid: 8 px; scale steps: 10,12,14,16,20,24,32,40,48
- Leading: 1.25x–1.5x; block spacing: 8–16 px
- All caps for IDs; prefixes and tags

Typesetting rules
- Tight tracking for uppercase labels; body copy readable tracking
- Alignment: left-aligned; mono columns for data blocks

Labeling conventions
- IDs: U-XYZ-0001, D-IT-002, O-USA-001
- Prefixes with fixed length and zero padding
