title: Layout System
---

Modular grid
- Base unit: 8 px; 12-column desktop grid; gutters 16 px
- Breakpoints: xs 0, sm 640, md 768, lg 1024, xl 1280
- Container max width: 1120 px

Aspect ratios
- Cards: 4:3 or 16:9 for diagrams

Composition patterns
- Label blocks, technical cards, badge areas
- Dividers: 2 px neutral lines

Spacing rules
- Padding: 8,12,16,24,32 px; group blocks with 8 px rhythm
