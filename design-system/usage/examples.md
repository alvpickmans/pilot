Usage Examples
==============

- Poster sample: industrial lab safety poster with a 12-column grid, using logo primary, title in monospace, body text in sans, and a row of redacted style stripes along the bottom.
- Badge mockups: circular seal with inner gear motif; outer ring bears the organization name; one-line status label inside.
- Interface panels: technical card with header (U-IT-003), body with 2-column data grid, right-aligned action bar.
- Device/Packaging label: 6x4 card on a device with unit id, origin, and hazard data; uses pattern background with subtle dot texture.
- Document label: header with document number prefix; data lines; a small barcode-like ticks area for machine readouts.

Notes
- All visuals follow the 8 px grid and 2 px stroke rules; ensure legibility on print and screen; ensure contrast compliance.
- Use the included tokens for colors, typography, and spacing for consistent rendering.
