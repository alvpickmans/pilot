Color System
============

Theme: Monochrome industrial neutral with optional accents

Primary Neutrals (Grayscale)
- 0: #0A0A0A (near black)
- 2: #1F1F1F
- 4: #333333
- 6: #5A5A5A
- 8: #7F7F7F
- 9: #A6A6A6
- 10: #FFFFFF (white)

Accent (optional secondary, used sparingly)
- Blue-gray: #2B6D9B (for highlights and data readouts)
