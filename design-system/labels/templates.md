Templates: Labels & Information Blocks
=====================================

- Unit ID Block (compact data card)
  - Header: UNIT ID
  - Body: ID: U-<DEPT>-<SERIAL>
  - Subfields: Unit, Origin, Manufacturer, Date
  - Visual: bold monospace ID, light secondary text for metadata

- Clearance Level Block
  - Header: CLEARANCE
  - Content: levels (UNCLASSIFIED, CONFIDENTIAL, SECRET, TOP SECRET)
  - Visual: striped bar indicating level intensity

- Department Label
  - Name: <DEPT> Department
  - Code: D-<CODE>
  - Visual: small emblem with stripe border

- Origin / Manufacturing Tag
  - Content: Origin: <COUNTRY> / Manufacturer: <NAME>
  - Visual: data barcode-like ticks at bottom edge for machine readout

- Hazard / Biosafety Info
  - Content: Hazard: <level> / Biosafety: <BSL>
  - Visual: hazard icon + color-coded flag (monochrome)

Typography & Alignment Rules
- All caps for IDs and prefixes; body copy uses sentence case
- Alignment: left-aligned fields; ID column on the left, metadata on the right
