Design System Style Guide (Summary)

Color
- Primary: #111111; accent: #4A7BAF; backgrounds: #FFFFFF, #F5F5F0, #EDEDED
- E‑Ink mode: tones 0/1/2 => #000000, #7F7F7F, #FFFFFF

Typography
- Primary sans: Inter; Secondary monospace: JetBrains Mono
- Scale: Display/Headline > Body > Caption; eInk optimized with generous leading

Layout
- Baseline: 8px; Safe zones: 16px; 12/8 grid rhythm
- Breaks: deliberate grid breaks for calibration motifs

UI Components
- Outline buttons, rectangular toggles, utilitarian tabs, metric cards, hairline dividers
- Charts: bar graphs, line telemetry, numeric readouts
- Calibration panels and system status modules

Icons
- Monochrome, 1–2 px strokes; scalable SVGs; crosshairs, scanlines, ticks

Motion
- Optional motion minimal; if used, single axis; provides a stepped feel for e‑ink fallbacks
